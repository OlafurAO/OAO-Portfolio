METIS – Valorant AI Coaching Companion

Project Duration: 2024 - 2025

View Discord App (Sunsetting in July 2025)

Overview

Developed a Discord bot designed as an AI coaching companion for Valorant players. Once linked with a user's Riot account, the bot automatically delivers post-match summaries and performance analyses, comparing the user's stats against both personal and rank-based averages.

Features

  • Retrieve average performance stats
  • Request agent-specific lineups for selected maps and positions (e.g., A/B/Mid, with ability filters)
  • Chat directly with an AI agent powered by ChatGPT, integrated using Microsoft Semantic Kernel

Technical Overview

The backend leverages Semantic Kernel plugins to fetch personalized user data, game metadata (agents, maps, weapons), and perform natural language understanding via a vector database for enhanced contextual relevance.

The Esports Coaching Academy Platform

Project Duration: 2022 - 2025

Visit Platform (Sunsetting in July 2025)

Overview

Initially led frontend development and UI/UX design for a web platform hosting game-specific esports training content. After the CTO stepped back from day-to-day responsibilities, I assumed full control of the platform’s tech stack—including backend development and database architecture.

Features

  • Hosts both curated and user-generated training content, with access governed by subscription tier
  • Coaches can form organizations with multiple locations, inviting other coaches to collaborate privately
  • Group management tools allow coaches to assign exercises and schedule sessions for students
  • Integrated calendar system to track session dates across an organization’s locations and coaches
  • Rich text editor powered by TinyMCE for content creation
  • Subscriptions and payments handled via Stripe

Technical Overview

The backend leverages Semantic Kernel plugins to fetch personalized user data, game metadata (agents, maps, weapons), and perform natural language understanding via a vector database for enhanced contextual relevance.

  • Designed and implemented frontend using React.js, TypeScript, Remix, and Tailwind CSS
  • Led UI/UX design, focusing on responsiveness, coach workflows, and intuitive navigation
  • Later expanded to backend ownership: built API routes, designed and optimized MySQL schema, and managed cloud infrastructure
  • Integrated Stripe, Sendgrid, Sentry, and deployed via Vercel and Azure

Roobi - Mobile Puzzle Game

Project Duration: 2021 -

Available on Google Play

Overview

Designed, developed, and released a mobile physics-based puzzle game independently using the native Android Java SDK. Built entirely from scratch, including code, game design, art, and music.

Highlights
